# 各プラグインの設定(vimscript)
プラグインの設定はこのディレクトリ内に配置する。
基本はluaで書く方針。
vimscript->luaの移行ができてないやつは`autoload/plugins.vim`に集めてる。
vimscriptで書いた設定はlua側をエントリーポイントにしてlua側から呼ぶ方針(基本的には)。

## 関数の命名規則
- "hookの種類_プラグイン名"とする
- ハイフンはアンダーバーに変更
- 以下は省略する。ただし、省略するとわかりづらい場合は除く。
  - "vim-"
  - "nvim-"
  - ".vim"
  - ".nvim"
  - ".lua"
